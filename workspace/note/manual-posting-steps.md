# note手動下書き投入手順

自動投入が使えない場合は、以下の手順で下書きを作成してください。

1. ブラウザで `https://note.com` を開く。
2. ユーザー自身でログインする。
3. 新規記事作成画面を開く。
4. `workspace/note/note-draft.md` のタイトルと本文を貼り付ける。
5. 有料部分にしたい範囲を、人間が確認してnote側の設定で調整する。
6. 下書き保存する。
7. 公開ボタンは押さない。

注意:

- ログイン情報、Cookie、セッション情報、トークンをファイルに保存しない。
- 価格設定や有料設定は勝手に確定しない。
- 公開前に、権利、表記、誇大表現、Higgsfield課金条件の記載を確認する。
