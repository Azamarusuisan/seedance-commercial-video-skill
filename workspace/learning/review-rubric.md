# Review Rubric

- First 2 seconds have a silent visual hook.
- Product/category is visible by the target second.
- Composition is preserved from Blender.
- Blender cheap-CG look is not preserved.
- Material quality is photoreal enough for the project.
- Lighting reads as photographic, not drawn graphics.
- Face/product distortion is acceptable or blocked.
- Text, subtitles, CTA, and claims are handled in post-production.
- CTA and title area are clear if needed.
- LP/promise/compliance is consistent for ads.
- Rights/compliance risk is known.
- Product/character memory is preserved across shots.
- Review identifies whether a new failure pattern candidate exists.
