# TikTok Theater Cast 60s Script

Purpose: Use the generated cast library as a small recurring troupe for TikTok/Shorts story videos. This is a reusable structure, not a final brand ad.

## Core Idea

Title: `終電後、全員が同じ地図を持っていた`

Format:

- Length: 60 seconds
- Video build: 4 x 15s Seedance visual-only clips
- Voice: Japanese narration via Higgsfield ElevenLabs or approved TTS
- Subtitles: edited Japanese telop after voice timing
- Music/SFX: added in edit, not required inside Seedance
- Primary platform: TikTok / Shorts, 9:16
