# TikTok Spark vs Non-Spark Decision

## Spark Ads

Use when:

- Strong organic post already exists:
- Creator/brand post should accumulate engagement:
- Social proof/comment thread matters:
- Poster/rights owner has approved paid usage:
- Spark authorization/account link is ready:

Risks/checks:

- Usage scope:
- Creator/face/voice rights:
- Music rights:
- AIGC disclosure:
- Comment moderation owner:

## Non-Spark Ads

Use when:

- Ad-only creative:
- Drafts should not stay as organic posts:
- Many AI-generated variants needed:
- LP/CTA/copy must be tested quickly:
- Spark authorization is not ready:

Decision:
`Spark Ads / Non-Spark Ads / both / undecided`

Reason:

Human approval owner:
