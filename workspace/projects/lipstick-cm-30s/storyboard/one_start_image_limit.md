# One Start Image Limit

Compatibility alias for `one-start-image-policy.md`.

Seedanceに渡すprimary imageは、1クリップにつき1枚の承認済みphotoreal key visualだけにする。

## Rules

- Do not collage Blender, product, lips, cast, and mood references into one flat source image.
- Use Blender only as composition source: camera, scale, placement, continuity.
- Use GPT Image / image generation to create the photoreal storyboard frame first.
- Use only the human-approved photoreal key visual as Seedance primary image.
- Keep lips/product/support references separate in manifests and prompts.
- If one start image cannot express the idea, split the shot or use start/end references only after model support and permissions are confirmed.
- If the same concept fails repeatedly, stop and redesign the storyboard instead of adding more prompt text.
