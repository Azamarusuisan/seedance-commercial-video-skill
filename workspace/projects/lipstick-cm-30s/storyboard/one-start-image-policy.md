# One Start Image Policy

Seedanceの入力が1枚だけになる場合でも、複数の参照素材を1枚に貼り合わせない。

## Rule

- Blender render is composition only.
- Product reference, lips reference, cast reference, and style reference keep separate roles.
- Photoreal key visual generation may use multiple references with explicit roles.
- Seedance receives only the approved final key visual as its primary image.

## Do Not

- Do not paste product and lips into one flat composite.
- Do not use Blender render as the primary Seedance image.
- Do not use a lips crop as an identity shot.
- Do not proceed after repeated mismatch by adding more text to Seedance only.

## If One Start Image Is Too Limiting

Split the concept:

1. product hero key visual
2. lips macro key visual
3. final packshot key visual

Then make shorter Seedance clips or request model support for start/end references. If the same concept fails three times, stop and redesign the storyboard.
